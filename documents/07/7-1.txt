----- 7-1.  Выбор и фильтрация элементов формы ------------

1.	Многие элементы форм создаются с помощью одного и того же тега input. 
	Поэтому разработчики создали специальные фильтры для этого тега.
	Например, для выбора всех элементов checkbox из форм нужно просто - $(':checkbox')
	А для выбора всех тектовых полей из конкретной формы на странице - $('#my_form :text')

------------ Создание выборок из элементов формы -----------
:checkbox	- выбор всех элементов checkbox из форм * 
:radio		- выбор всех radio кнопок из форм *
:submit		- выбор всех кнопок submit *
:reset		- выбор всех кнопок сброса 
:hidden 	- выбор всех полей типа "Скрытый" <input type="hidden"..
:file 		- выбор всех кнопок загрузки файлов
:button		- выбор всех кнопок * (проверить после добавления такой кнопки!)
:text		- выбор всех тектовых полей *
:password	- выбор всех полей для ввода пароля
:image 		- выбор всех кнопок в виде картинок
:input		- выбор всех элементов, которые влияют на ввод данных в форме *
			---
:checked 	- выбор активных элементов *(!Отправить заявку)
:selected 	- выбор активного элемента select *(!Отправить заявку)
