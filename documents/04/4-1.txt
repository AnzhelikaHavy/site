---------- Основы выборки элементов в jQuery:--------------

	УРАВНЕНИЕ: 			ОБЪЕКТ - СОБЫТИЕ - ДЕЙСТВИЕ
						ОБ'ЄКТ -  ПОДІЯ	 -   ДІЯ (методи)
						
	-	объект 			(отобрать элементы), 
	-	событие 		(повесить на отобранные элементы определенные события), 
	-	действие		(назначить действия для этих событий)

--------------  События --------------------------------
Браузер отслеживает все события пользователя: перемещение курсора мыши, 
прокрутка страницы, щелчок мыши... 
И браузер проверяет: нет ли каких-либо действий привязанных к этому событию.

-------------- Список СОБЫТИЙ --------------------------
Какие события отслеживает браузер? 
Например mouseover - наведение мыши на элемент (первый мотоцикл)
Всплывание событий: Событие срабатывает сразу для нескольких элементов:
для общего блока div, для блока div с мотоциклами, для самого мотоцикла.
********************************************************  
mouseover	- наведение мыши на элемент 
mouseout	- увод курсора мыши с элемента
click		- левый клик
dblclick	- двойной щелчок

mousemove	- движение курсора мыши
mousedown	- нажали левую кнопку мыши но еще не отпустили
mouseup		- отпустили кнопку мыши
...

-----------   В традиционном javascript ------------

Объект: первый мотоцикл. Событие: клик мыши. Действие: Вывести ширину картинки.

1.	Выбор объекта:
	-	Объект: document  
	-	Метод: getElementById (выбор элемента по его id: 'img_1')
	-	Запишим выбранный объект в переменную myMoto
	
2.	Назначим действие. Действие следует записать в виде функции motoClick()
	-	Выводим с помощью alert: "Ширина данной картинки: " + myMoto.width (ширину картинки) 
		
3.	Вызов функции организуем на событие onclick
	-	myMoto.onclick = motoClick;

Недостатки: Работает по-разному в разных браузерах. 
			Для каждого элемента нужно прописывать свой обработчик событий.
