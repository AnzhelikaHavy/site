******************   32. Изменение размеров элементов   **********************

	Интерактивный элемент: Resizable.	Exemples: Max/min size.
	
	Элемент с идентификатором mytextarea. Добавим возможность изменения размера.
	
1.	В индексном файле: убрать cols="46" rows="20". (оставить 45,5)

2.	В таблице стилей: Зададим размеры 380*150

3.	Скрипт: Добавим возможность изменения размеров только по высоте. Копируем код.

4.	Скрипт: Изменяем пределы:
	максимум:	420*375
	минимум:	150*375
