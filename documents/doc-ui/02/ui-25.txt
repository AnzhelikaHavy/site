*************************   19. Колличество и цена   ***************************

1. 	Изначально в html коде у нас уже есть блок для будущего расчета колличества
	и цены шлемов, уложенных пользователем в корзину. Внутри этого блока есть и
	форма с кнопкой для отправки формы. Но этот блок первоначально скрыт.
	Для расчета нужны два счетчика. Перед входом в код для корзины определим 
	две переменные helmetCount и summa. Обе переменные нужно сначала обнулить.

var helmetCount = 0;
var summa = 0;
	
2.	Действия пользователя. Шлем выносится с полки в корзину.
	Для корзины происходит событие drop:  		В коде корзины добавляем:
	-	Увеличиваем колличество на 1   (helmetCount++)
	- 	Если счетчик становится больше 0, то блок #myclear нужно показать show()
			helmetCount++;
			if (helmetCount > 0) {			$('#myclear').show();		}	
	
3.	Обратные действия. Пользователь выносит шлем из корзины обратно на полку.
	Для полки происходит событие drop:  		В коде полки добавляем:
	-	Уменьшаем колличество на 1   (helmetCount--)
	- 	Если счетчик становится равен 0, то блок #myclear снова скрывается hide()
			helmetCount--;
			if (helmetCount == 0) {			$('#myclear').hide();		}	
	
4.	Запишем результат с помощью метода text()
	Сначала отберем элемент с идентификатором helmetsCount, а внутри него тег strong
			$('#helmetsCount strong').text(helmetCount);
	
5.	Для расчета суммы нам нужны цены на шлемы. 
	Для простоты эти цены просто указаны в атрибуте title блоков со шлемами.
	Текущий шлем, который принес пользователь, у нас лежит в переменной helmet.
	Применим к этой переменной метод attr() для того чтобы узнать стоимость шлема.
	И запомним это значение в переменной summa. 
			summa = helmet.attr('title');
	
	Запишем результат в  соответствующий блок 
			$('#helmetsSumm strong').text(summa);
		
	В атрибуте title записано текстовое значение. Его нужно преобразовать в число.
	Используем javascript метод parseInt() для преобразования текста в число.
			summa = parseInt(helmet.attr('title'));
		
	Вместо функции можно использовать для преобразования оператор +
			summa = +helmet.attr('title');
			
	Если пользователь внес в корзину несколько шлемов, их цены должны складываться, 
	а переменныя summa должна накапливать сумму цен +=
			summa += +helmet.attr('title');
			
6. 	Обратные действия. Пользователь выносит шлемы из корзины на полку.
			summa -= +helmet2.attr('title');
